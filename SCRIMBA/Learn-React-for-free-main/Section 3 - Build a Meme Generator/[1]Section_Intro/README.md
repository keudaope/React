# Section Intro
```Static Web Sites``` vs ```Dynamic Web Apps```?
- ```Static Web Pages``` are ```read-only``` meaning no changes to data (eg: Blogs, News sites, Recipes, etc.)
- ```Dynamic Web Apps``` are ```read-write``` meaning the ability to change data, highly interactive, and capable of displaying your data (eg: Bank Website, AirBnb, etc.)
